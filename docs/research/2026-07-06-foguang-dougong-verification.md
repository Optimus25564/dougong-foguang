# 佛光寺东大殿外檐柱头铺作 · 复原核实与来源

> 对本项目 3D 复原模型所编码结构事实的核实（2026-07-06，多源检索 + 对抗式验证）。
> 建筑对象：山西五台山佛光寺东大殿，唐大中十一年（857 年）重建，现存规模最大、
> 保存最完好的唐代木构；四座唐构中唯一用七铺作者，为孤例。

## 关键修正：隔跳偷心造

梁思成《记五台山佛光寺的建筑》对外檐柱头铺作的逐跳记述：

> 一跳偷心 → **二跳施瓜子栱、慢栱承罗汉枋（计心）** → 三跳偷心 →
> 四跳令栱与耍头相交，令栱上施替木承橑檐槫。

即 **一、三跳偷心；二、四跳计心**——是「隔跳偷心造」，非全偷心。

**模型已据此补正**：在第二跳（华栱二跳头 / `jiaohudou-2`）上补入计心重栱层：
`guazigong`(瓜子栱) → `sandou-5/6` → `mangong`(慢栱) → `sandou-7/8` → `luohanfang`(罗汉枋)。
部件总数 19 → 26；散斗由 4 增至 8（泥道 2 + 瓜子/慢栱 4 + 令栱 2）。

## 存疑 / 待一手核实

- 令栱是否省齐心斗：需查梁思成原始实测图或陈明达《营造法式大木作研究》原文。模型暂省并在注释标注存疑。
- 耍头断面：批竹昂形 vs 翼形，来源并存，宜以一手图证确认。
- 材广/分°精确值（30 vs 31.5cm）：精确建模宜以陈明达/傅熹年实测原文为准。

## 总判

「七铺作双杪双下昂」的铺作等级与出跳构成**正确无误**，且为唐构孤例。经本次核实，
模型已由「全偷心」改正为「隔跳偷心造」、移除无据的「柱头枋隐出栱」、并将「令栱省齐心斗」
标为存疑。除上述已处理项外，未发现与实物不符之处。
